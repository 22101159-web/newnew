================================================================================
GABRIELLE CUSTODIO EVENT STYLING - FULLSTACK TRANSFER GUIDE (NODE.JS)
================================================================================

This guide explains how to set up the project on your local PC using the
Node.js (Express) backend.

--------------------------------------------------------------------------------
1. PREREQUISITES
--------------------------------------------------------------------------------
Before starting, ensure your PC has:
- Node.js (v18 or higher): https://nodejs.org/

--------------------------------------------------------------------------------
2. EXPORTING THE PROJECT
--------------------------------------------------------------------------------
- In AI Studio, go to Settings -> Export as ZIP.
- Extract the ZIP file into a folder on your PC (e.g., C:\Projects\event-styling).

--------------------------------------------------------------------------------
3. PROJECT STRUCTURE OVERVIEW
--------------------------------------------------------------------------------
Your project is built with a unified Node.js architecture:
- /root: React Frontend, Configuration, and the server (server.ts).
- /server: Backend logic, database setup, and authentication.

--------------------------------------------------------------------------------
4. INSTALLING DEPENDENCIES
--------------------------------------------------------------------------------
Open your terminal in the root folder and run:
   npm install

This will install all frontend and backend dependencies at once.

--------------------------------------------------------------------------------
5. CONFIGURATION (.env File)
--------------------------------------------------------------------------------
Create a file named .env in the root directory and add your keys:

GEMINI_API_KEY=your_gemini_key
VITE_CLOUDINARY_CLOUD_NAME=your_cloudinary_name
VITE_CLOUDINARY_UPLOAD_PRESET=your_preset
SECRET_KEY=your_secure_random_string_here

--------------------------------------------------------------------------------
6. RUNNING THE APPLICATION
--------------------------------------------------------------------------------
In the root folder, run:
npm run dev

This will start the integrated server that handles both the API and the 
React frontend.

Open http://localhost:3000 in your browser.

--------------------------------------------------------------------------------
7. AUTHENTICATION & DATABASE
--------------------------------------------------------------------------------
- The system uses SQLite (`database.db`) locally.
- On the first run, it will automatically create the database and seed an 
  initial admin:
  Email/Username: admin@example.com
  Password: Admin123

- JWT (JSON Web Tokens) are used for secure session handling.
