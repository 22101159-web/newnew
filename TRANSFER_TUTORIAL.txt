TRANSFERRING YOUR PROJECT TO YOUR LOCAL MACHINE
=================================================

This is a comprehensive guide to transferring your project from this environment to your local machine. This guide covers how to set up both the React/Vite frontend and the Python FastAPI backend using VS Code.

Since you are exporting this project directly from AI Studio, you don't need to manually recreate the folders or copy-paste code! The ZIP file you download will already contain everything.

Phase 1: Download the Code
--------------------------
1. Click the "Settings" menu in the top right corner of AI Studio.
2. Select "Export to ZIP" (or export it to GitHub and clone the repository).
3. Extract the downloaded ZIP file to somewhere on your computer.

Phase 2: Preparations
---------------------
Before you start, ensure you have these tools installed on your computer:
1. Node.js (v18 or higher) - Includes "npm"
2. Python 3 (v3.9 or higher)
3. pip (Python package installer - comes with Python usually)

Phase 3: Opening in Visual Studio Code
--------------------------------------
1. Open VS Code.
2. Go to File > "Open Folder..." and select the extracted folder.
3. Open a new built-in terminal in VS Code (Terminal > New Terminal, or press `Ctrl + \``).

Phase 4: Install Dependencies
-----------------------------
In the VS Code terminal, run the following command to install the frontend Node modules:

npm install

Phase 5: Running the Full-Stack App
-----------------------------------
Once NPM finishes installing, start the application by running:

npm run dev

What this command does automatically behind the scenes (via server.ts):
1. It installs the Python requirements (`pip install -r backend/requirements.txt`).
2. It starts the Python FastAPI server on port 8000.
3. It starts the Vite React frontend.

Once the terminal says both servers are started, open the provided localhost link (usually http://localhost:5173 or 3000) in your browser.
