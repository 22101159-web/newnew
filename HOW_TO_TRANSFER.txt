HOW TO TRANSFER THIS PROJECT TO YOUR LOCAL PC (WINDOWS/MAC/LINUX)

This is a Full-Stack application using Vite (Frontend) and Python FastAPI (Backend) with SQLite.

Prerequisites:
1. Install Node.js (Version 18 or higher) from https://nodejs.org/
2. Install Python (Version 3.10 or higher) from https://www.python.org/
3. A code editor like VS Code.

Common Issues & Requirements:
- Python Environment: Ensure `python3` and `pip` are in your PATH.
- SQLite: The application uses SQLite, which is built into Python. No separate installation is needed for the database engine.

Steps to Setup:
1. Export the project:
   - Go to the Settings menu in AI Studio.
   - Choose "Export to ZIP" or "Export to GitHub".
   - Extract the ZIP file to a folder on your computer.

2. Install Dependencies:
   - Open a terminal/command prompt in the project folder.
   - Run the following command to install Frontend and Manager dependencies:
     npm install
   - The Python backend dependencies will be automatically installed when you run `npm run dev`, or you can manually install them using:
     pip install -r backend/requirements.txt

   This will install all necessary packages including:
   - Frontend: React 19, Vite 6, Tailwind CSS 4, Framer Motion, Lucide Icons, FullCalendar.
   - Backend (Python): FastAPI, SQLAlchemy (ORM), JWT, Bcrypt, Uvicorn.
   - Manager: Express (used to proxy requests and run both servers together).

3. Environment Variables:
   - Copy '.env.example' to a new file named '.env'.
   - In your '.env' file, set APP_URL="http://localhost:3000".
   - Note: Do NOT delete '.env.example'. It is a template used to track which variables are required for the project without storing actual secrets.

4. Run the Development Server:
   - Run the following command:
     npm run dev
   - This starts both the Node.js manager (Port 3000) and the Python FastAPI backend (Port 8000).
   - Access the app at http://localhost:3000

5. Database:
   - The application uses SQLite. The 'database.db' file will be automatically created and seeded in the root directory when you first run the server.
   - Default Admin Password: Admin123

6. Production Build:
   - To build for production:
     npm run build
   - To start the production server:
     npm start

Enjoy your application!
